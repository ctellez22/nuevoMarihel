⚡ SOLUCIÓN EN 3 PASOS
────────────────────────────────────────────────────────────────────────────────

PASO 1: OBTÉN LA IP
  ./scripts/find-printer-ip.sh
  (O: Panel de control impresora → Network → Anota IP)

PASO 2: VERIFICA CONEXIÓN
  nc -zv 192.168.1.100 9100
  (Debe decir: "Connection succeeded" ✅)

PASO 3: EJECUTA LA APP
  java -Dprinter.ip=192.168.1.100 -jar marihel.jar
  (¡Crea una joya y debería imprimirse!)

🎯 CÓMO FUNCIONA AHORA
────────────────────────────────────────────────────────────────────────────────

ANTES:
  ❌ Solo intentaba comando 'lp'
  ❌ Si no tenías CUPS → error inmediato
  ❌ Sin alternativa

AHORA:
  1️⃣  Intenta Socket TCP → Envía ZPL directo a IP:9100 (RÁPIDO)
  2️⃣  Intenta /usr/bin/lp → Usa CUPS si existe
  3️⃣  Intenta lp en PATH → Fallback
  4️⃣  Si todo falla → Muestra instrucciones claras
